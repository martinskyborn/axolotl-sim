# Axolotl Sim

Ett svenskt mobilspel för Safari på iPhone. Tryck för att simma uppåt, samla stjärnor och undvik tången.

Spelet är en statisk PWA byggd med HTML, CSS och Canvas. Det kan köras lokalt eller publiceras direkt på Vercel.

## Kör lokalt

```bash
python3 -m http.server 5173
```

Öppna sedan `http://localhost:5173`.
